# SeedValidator Finance - User Guide

## Welcome

SeedValidator Finance is an AI-powered FP&A (Financial Planning & Analysis) platform designed for startups. This guide will help you get started and make the most of the platform.

---

## Getting Started

### 1. Sign Up / Log In

1. Visit the application at your deployed URL
2. Click **Sign Up** to create a new account
3. Or use **Google/GitHub** OAuth for quick sign-in

### 2. Dashboard Overview

After logging in, you'll see the **Financial Command Center** with:

- **Key Metrics** - Cash balance, burn rate, runway, MRR
- **Burn Rate Trend** - Visual chart of your spending patterns
- **Recent Transactions** - Latest financial activity
- **AI Insights** - Smart recommendations

## Integrations

### Connect Your Bank

1. Go to **Integrations** or click **Connect Bank**
2. Use Plaid Link to securely connect accounts
3. Transactions sync automatically

### Available Integrations

| Service | Purpose              |
| ------- | -------------------- |
| Plaid   | Bank account syncing |
| HubSpot | CRM pipeline data    |
| Stripe  | Payment processing   |
| OpenAI  | AI-powered insights  |

## Tips for Success

1. **Connect all bank accounts** for complete visibility
2. **Review forecasts weekly** to catch trends early
3. **Set up alerts** for burn rate and runway thresholds
4. **Export reports monthly** for investor updates
